# 📸 QR Code Generator 🎉

Welcome to the **QR Code Generator**! 🚀 This project lets you generate QR codes with ease using Python.
Whether you need QR codes for URLs, text, or anything else, this tool has got you covered! ✅

## 🌟 Features
- Generate QR codes in seconds ⏱️
- Save QR codes as PNG images 📁
- Customizable input for any type of data ✨
- Lightweight and easy to use 🪶

## 📦 Requirements
Before running the program, make sure you have Python 3.x and the required libraries installed:

- Python 3.x 🐍
- `qrcode` 📋

You can install the dependencies by running:

```bash
pip install qrcode[pil]
```

## 💡 How It Works
- The script takes your input (like text or a URL) and generates a QR code using the `qrcode` library. 🔍
- The generated QR code is saved as a high-quality PNG image. 🖼️
- You can scan it with any QR code reader to view the encoded data. 📱
